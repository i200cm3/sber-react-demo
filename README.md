# sber-react-demo

Copy your token from: https://smartapp-studio.sberdevices.ru/ to ".env" (REACT_APP_SBER_TOKEN)

Add your smart app name to ".env" (REACT_APP_SMARTAPP)

Install dependency:
```
npm install
```

Local run and debug:
```
npm run start
```

Build for production:
```
npm run build
```
